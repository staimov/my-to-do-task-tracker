# my-to-do-task-tracker

![screenshot](/src/imgs/screenshot.jpg?raw=true)

## Usage

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the JSON server (http://localhost:5000)

```
npm run server
```

### To build for production

```
npm run build
```
